# Enrutador

## Descripción
El proyecto de *Enrutador* tiene como función poder marcar en un mapa se tipo OSM ([Open Street Map](http://openstreetmap.org/)) registros de contactos, en zonas geográficamente personalizadas y dentro de este poder colocar sus datos.

1.- Internos de la tarjeta de contacto.
- Nombre completo.
- Dirección Física.
- Numero teléfonicos.
- Estado del contacto.
- Tipo de contacto.
- Enlace a otros contactos.
- Notas.
- Fotos de perfil.

2.- Externos del Punteo
- Geolocalización.
- Compartir
- Enrutador.
- Agendar.
- Estado de sincronización
- Compartir
La función principal de esta aplicación ubicar al contacto en un mapa de manera más eficiente, todo manejado desde la nube como respaldo usando la plataforma de firebase de Google mediante un sistema de pendientes manejado por un sistema interno de administrador.
