# Enrutador

## Descripción
El proyecto personal del enrutador tiene como función poder marcar en un mapa se tipo OSM ([Open Street Map](http://openstreetmap.org/)) registros de contactos, y dentro de este poder colocar sus datos.
- Nombre ccompleto
- Dirección Física
- Numero teléfono
- Número Alternativo
- Estado del contacto
- Tipo de contacto
- Enlace a otros contactos
- Notas
Mas...

La función principal de esta aplicación ubicar al contacto en un mapa de manera más eficiente, todo manejado desde la nube como respaldo usando la plataforma de firebase de Google mediante un sistema de pendientes manejado por un sistema interno de administrador.
